📊 专题一：数据、信息与人工智能

1. 数据定义与特征

数据是对客观事物的符号表示，如图形符号、数字、字母等。单纯的数据没有意义，经过解释才变得有意义。

数据特征：客观性、普遍性、多样性、感知性。表现形式可以是文字、图形、图像、音频、视频等。

2. 信息的概念与特征

信息的特征：载体依附性、时效性、共享性、可加工处理性、真伪性、价值性。

重要概念：信息不能独立存在，必须依附于载体。信息可以脱离它所反映的事物被存储、保留和传播，但不能脱离载体。

3. 数据、信息与知识的关系

信息是数据经过存储、分析及解释后所产生的意义。

数据 → 储存、分析及解释 → 信息 → 分析、判断、确认、归纳 → 知识

4. 数据采集与数字化

模拟信号以连续变化的物理量存在，数字信号在取值上是离散的、不连续的信号。信息必须经过数字化之后计算机才能进行存储和处理。

数字化过程：采样 → 量化 → 编码

5. 进制转换

二进制转十进制：按权展开求和

十进制转二进制：除2取余，逆序排列

6. 字符、声音、图像及视频编码

ASCII码：128个字符，1字节中的低7位编码，范围00-7F

汉字编码：外码、交换码、机内码、字形码

⚙️ 专题九：数据结构与算法

1. 算法效率的概念及计算方式

时间复杂度：O(1) < O(log n) < O(n) < O(n²) < O(2ⁿ) < O(n!)

2. 排序算法

冒泡排序：相邻元素比较交换

选择排序：每次选择最小元素

插入排序：将元素插入已排序序列

# 冒泡排序 def bubble_sort(arr): n = len(arr) for i in range(n-1): for j in range(0, n-i-1): if arr[j] > arr[j+1]: arr[j], arr[j+1] = arr[j+1], arr[j] return arr # 测试 arr = [64, 34, 25, 12, 22, 11, 90] print(bubble_sort(arr.copy()))

3. 查找算法

顺序查找：逐个比较

二分查找：要求数据有序，时间复杂度O(log n)

# 二分查找 def binary_search(arr, target): left, right = 0, len(arr)-1 while left <= right: mid = (left + right) // 2 if arr[mid] == target: return mid elif arr[mid] < target: left = mid + 1 else: right = mid - 1 return -1 # 测试 arr = [1, 3, 5, 7, 9, 11, 13] print(binary_search(arr, 7)) # 3 print(binary_search(arr, 10)) # -1
